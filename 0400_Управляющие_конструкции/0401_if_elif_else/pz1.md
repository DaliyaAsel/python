# Практическое задание: Определение оценки по числовому баллу

## Цель

Научиться использовать условные операторы `if-elif-else` для ветвления логики программы, а также реализовать валидацию ввода пользователя.

## Задание

Создайте программу, которая запрашивает у пользователя числовой балл и выводит соответствующую словесную оценку по следующей шкале:

- **90–100** → "Отлично"
- **75–89** → "Хорошо"
- **60–74** → "Удовлетворительно"
- **0–59** → "Неудовлетворительно"

### Шаги выполнения:

1. Запросите у пользователя ввод числа с помощью `input()`
2. Преобразуйте введённое значение в целое число (`int`)
3. Реализуйте проверку корректности ввода:
   - Если число **меньше 0**, выведите сообщение: `"Ошибка: оценка не может быть отрицательной"`
   - Если число **больше 100**, выведите сообщение: `"Ошибка: оценка не может быть больше 100"`
4. Для корректных значений (от 0 до 100 включительно) определите оценку с помощью `if-elif-else` и выведите результат

### Примеры работы программы:

```
Введите ваш балл (0-100): 87
Хорошо
```

```
Введите ваш балл (0-100): -5
Ошибка: оценка не может быть отрицательной
```

```
Введите ваш балл (0-100): 150
Ошибка: оценка не может быть больше 100
```

- [ ] Программа использует `input()` для получения данных от пользователя

## Советы по улучшению работы:

1. **Проверьте граничные значения** — самый частый источник ошибок. Убедитесь, что балл 60 выводит "Удовлетворительно", а 90 — "Отлично".

2. **Порядок условий важен** — при использовании `if-elif-else` проверки выполняются сверху вниз. Сначала проверьте ошибки (<0 и >100), затем уже основную шкалу.

3. **Запустите программу несколько раз** с разными значениями: положительными, отрицательными, больше 100, граничными (0, 60, 75, 90, 100).
